README

Run this as any Java file, but requires the command line argument of the plugin folder. (I.E. 'test')

Dropping new .jars (required to have a PluginImplementation class that implements the IPlugin interface)
will load them to the GUI.

Clicking on the elements on the List will run the plugin.

There are two sample jars, "helloworld.jar" and "stopwatchPlugin.jar"
